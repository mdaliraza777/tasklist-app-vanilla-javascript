# Tasklist-app-vanilla-javascript.
A feature-rich Task List application built with HTML, CSS, and Vanilla JavaScript featuring task priorities, due dates, search, filters, dark mode, and local storage etc.
# Advanced Todo App

## Features
- Add, Edit, Delete Tasks
- Complete / Incomplete Tasks
- Task Statistics
- Search Tasks
- Filter Tasks
- Priority Levels
- Due Dates
- Dark Mode
- Local Storage
- Responsive Design

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)
